# 🛶 Monsoon Dash

A cute pixel-art survival game: guide a paper boat through a flooded street as the monsoon rages. Dodge floating debris, car splash-waves, and half-submerged rickshaws. The harder it rains, the darker the sky gets and the stronger the currents push you.

**▶️ Play it live:** https://monsoon-dash.vercel.app/

## Features
- One-tap controls (works on mobile + desktop)
- Rising rain levels with day → dusk → night shift
- Lightning, thunder & dynamic weather
- Chai-cup collectibles and an umbrella shield power-up
- Near-miss bonuses, combos, and a saved high score
- Procedural ambient music (no audio files)
---
## Tech
HTML5 Canvas + vanilla JavaScript, in a single file. No dependencies.
---
## Run locally
Just open `index.html` in a browser — that's it.
